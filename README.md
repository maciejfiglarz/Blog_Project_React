## General info

this project is a blog application which allow users to publish their own post. 
	
## Technologies

Project is created with:
* JavaScript ES6
* SASS
* CSS3
* HTML5
* RWD
* JSON
* Webpack
	
## Setup
To run this project, install it locally using:


```
$ yarn install
$ yarn run serve
$ yarn run json:server

```
